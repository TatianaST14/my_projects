Проект: анализ воронки продаж и конверсий для автодилера. В репозитории находится Power BI отчёт с визуализациями ключевых метрик, воронкой конверсий и аналитикой продаж по моделям/каналам.

Файлы
- Анализ конверсий и продаж Автодилер.pbix — Power BI Desktop файл с полным отчётом.
- Анализ конверсий и продаж Автодилер.pdf — краткий аналитический отчёт и ключевые визуализации (PDF, можно просмотреть прямо на GitHub).
