# Аналитика онлайн-кинотеатра по каналам привлечения

Анализ онлайн-кинотеатра, предоставляющего к просмотру как фильмы и сериалы, так и мультфильмы. В течение месяца клиент провёл серию рекламных кампаний и теперь хочет оценить, какой вклад в итоговый трафик они внесли.
Данные о реакции людей на различные рекламные каналы собраны на разных этапах воронки продаж. Воронка состоит из пяти последовательных этапов: осведомлённость о бренде → интерес → рассмотрение бренда к покупке → желание купить → покупка.

## Содержание папки:
- README.md — этот файл.
- Объединение данных из разных источников.ipynb — Объединение данных из разных источников в один файл.
- Обработка данных и расчет фродовых пользователей.ipynb — Исправление некорректных данных, заполнение пропусков, анализ воронки продаж по каналам привлечения, определение степени сфальсифицированности данных
- Факторный и корреляционный анализ.ipynb - Факторный и корреляционный анализ, интерпретация результатов
- Визуализация данных.ipynb - Визуализация данных, интерпретация результатов
- Гипотезы и выводы.ipynb - Формулировка гипотез, анализ этапов воронки, выводы по каналам привлечения
