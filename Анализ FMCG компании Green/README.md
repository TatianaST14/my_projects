Green - крупная FMCG компания, производящая консервированную продукцию и продающая её на рынке через большое количество ритейлеров, которые напрямую закупают товар у компании и в дальнейшем развозят по своим торговым точкам.
Рынок консервированной категории состоит из нескольких брендов, которые являются вашими конкурентами. Каждый бренд имеет свой ассортиментный портфель, наименования которого распределены по группам в соответствии с весовым признаком. Например, ассортимент бренда «Green» имеет шесть групп от 100-199G до 600-899G, где в каждой весовой группе может находиться разное количество наименований. 

Также рынок консервированной категории представлен тремя каналами продаж: гипермаркетами (ГМ), супермаркетами (СМ) и минимаркетами (ММ). Ассортимент бренда «Green» продаётся во всех трёх каналах.
